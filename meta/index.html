Questions

What if I've never danced, or feel awkward moving?

That's exactly who this is for. Both courses start from zero — no rhythm, flexibility, or dance background needed. Most students start exactly where you are now.

Do I need a partner?

Not to start. Floorwork is entirely solo. The CI course introduces partner work, but you can begin with the solo elements and find someone to practice with as you go.

Which course should I start with?

No partner right now — start with Floorwork. It builds the foundation that makes everything else easier. Already have someone to practice with — go straight to CI.

Can I get both?

Yes — they complement each other directly. Many students do both.

How long do I have access?

Forever. One purchase, permanent access.
